Smart Mirror

After completing an object-oriented programming course, I decided to pursue a project to solidify some programming concepts. In the course's last unit, we learned about JavaFx GUI design and development, so I decided I wanted to incorporate that into the project. Since I am also a mechanical engineer interested in product design, I thought a smart mirror project would be a great combination of design and programming.

I learned a ton working on this project. The first is working with APIs. The weather and calendar widgets rely on obtaining data from the Dark Sky API and the Google Calendar API. The second is awareness of the program's target platform. Up to this point, I never thought about code compatibility. It took me only a weekend to physically build the mirror and write the program. However, it took me another weekend just to deploy the code to the raspberry pi. I discovered that some of the libraries I used were not compatible with the raspberry pi's lightweight arm architecture, which led to a decent amount of code rewrites.
