Inverted Pendulum

The inverted pendulum was initially an individual project for my graduate mechatronics class, however, I returned to this project after college to make improvements. The pendulum utilizes the concept of reaction wheel torque to stay upright. The states of the system are measured by an inertial meausurement unit (IMU) and an encoder. The IMU gives both acceleration and gyroscopic values. These values can be fused through a complementary filter to determine the pendulum's angle and velocity. The encoder records the velocity of the reaction wheel. Knowing these states, control of the pendulum angle is possible using the flywheel's reaction torque. The first iteration of this project utilized a LQR controller implemented on a TI MSP 432 microcontroller. Later iterations used a PD + motor feedback control design for improved performance. The most recent iteration utilizes a genetically tuned full state feedback controller. The newest design also incorporates automatic setpoint adjustment to mitigate improper calibration or a shift in mass.
